#### 3. Database Migration System
Created `runtime/ops/migrations.ts` with:
- Schema version tracking in `schema_migrations` table
- Up/down migration support with transaction safety
- Migration file generator
- Status reporting and rollback capabilities
- CLI tool (`scripts/migrate.ts`) for operational use

Commands:
```bash
npm run ops:migrate status    # Show migration state
npm run ops:migrate up        # Apply pending migrations
npm run ops:migrate down [n]  # Rollback n migrations
npm run ops:migrate create    # Generate migration file
```

#### 4. Backup & Restore System
Created `runtime/ops/backup.ts` with:
- SQLite VACUUM-based consistent backups
- Gzip compression support
- Automated scheduling with rotation
- Restore from compressed/uncompressed backups
- Backup size tracking and metadata
- CLI tool (`scripts/backup.ts`) for operational use

Commands:
```bash
npm run ops:backup create              # Create backup (compressed)
npm run ops:backup create --no-compress # Uncompressed backup
npm run ops:backup list                # List all backups
npm run ops:backup restore <file>      # Restore from backup
npm run ops:backup clean [keep]        # Delete old backups
```
